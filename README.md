# Cricket-Match-Recorder
Forever free, adless and open-source cricket score recorder

## Current objective:
Recording cricket scores of the games played with friends or otherwise... Under the following constraints:
- Single webpage: Once the site is loaded, it should be fully functional even without an active internet connection.
- Compatibility: Internet Explorer 11, All modern versions of web browsers like Google Chrome, Mozilla Firefox, Microsoft Edge etc.

## Current status:
This website is still under construction... contributions are welcome!
